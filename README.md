# Hand-Signs
Hand sign classification 

- Use CNN to classify Hand sign images into alphabetic [A,B,C..... Y]
- 'Z' and 'J' are not included due to moving gesture.

## Built With

- matplotlib
- NumPy
- Tensorflow, keras
