# Hand-Signs
Hand sign classification 

- Use CNN to classify Hand sign images into alphabetic [A, B, C..... Y]
- 'Z' and 'J' are excluded due to the moving gesture.

## Built With

- matplotlib
- NumPy
- Tensorflow(keras)
